信号与系统 课程知识总结

一、绪论 信号与系统基本概念

1. 信号分类

  • 按时间划分：连续时间信号、离散时间信号
  • 按幅值划分：模拟信号、数字信号
  • 按周期性划分：周期信号、非周期信号
  • 典型基本信号：阶跃信号、冲激信号、矩形脉冲、斜变信号

2. 信号基本运算

包含移位、反转、尺度变换、相加、相乘、微分、积分，是后续分析的基础。

3. 系统分类

  • 线性/非线性系统、时不变/时变系统
  • 因果/非因果系统、稳定/不稳定系统

二、连续时间信号与系统 时域分析

1. 卷积积分

卷积是时域分析核心，线性时不变系统的零状态响应等于激励信号与单位冲激响应的卷积。

卷积公式：$f(t) * h(t) = \int_{-\infty}^{+\infty} f(\tau)h(t-\tau)d\tau$

2. 系统响应分解

全响应 = 零输入响应 + 零状态响应。零输入响应由初始状态引起，零状态响应由外部激励引起。

三、傅里叶级数与傅里叶变换（频域分析）

1. 傅里叶级数

周期连续信号可分解为不同频率正弦/余弦信号叠加，分为三角形式与指数形式。

2. 傅里叶变换

实现信号从时域到频域的转换，用于分析信号频谱、系统频率特性。

正变换：$F(j\omega) = \int_{-\infty}^{+\infty} f(t)e^{-j\omega t}dt$
逆变换：$f(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty} F(j\omega)e^{j\omega t}d\omega$

3. 常用性质

线性、时移性质、频移性质、尺度变换、卷积定理（时域卷积对应频域相乘）。

四、拉普拉斯变换 复频域分析

1. 基本定义

拉普拉斯变换是傅里叶变换的推广，收敛域更广，适用于求解电路、系统暂态响应。

单边拉普拉斯变换：$F(s) = \int_{0}^{+\infty} f(t)e^{-st}dt$

2. 核心应用

将时域微分、积分运算转化为复频域代数运算，大幅简化高阶系统方程求解。

3. 极点与零点

零极点分布决定系统时域特性、稳定性与响应形式。

五、离散时间信号与Z变换

1. 离散信号基础

典型信号：单位序列、单位阶跃序列、矩形序列、正弦序列。

2. Z变换

离散系统的核心分析工具，地位等价于连续系统的拉普拉斯变换。

3. 离散卷积

离散系统零状态响应为激励与单位响应的离散卷积和。

六、系统频率响应与滤波基础

系统频率响应描述系统对不同频率分量的幅度、相位传输特性。根据频率特性可分为低通、高通、带通、带阻滤波器，是工程应用重点。

七、公式汇总与典型考点

  • 1. 冲激信号的筛选性质（必考基础题）
  • 2. 卷积计算、分段卷积题型
  • 3. 三大变换（傅里叶、拉普拉斯、Z变换）正逆变换及性质应用
  • 4. 系统线性、时不变、因果、稳定性判断
  • 5. 零极点分析、频率响应求解

八、错题与学习总结

本课程难点集中在变换性质混用、卷积分段计算、零极点分析。学习核心思路：时域看运算，频域看变换，复频域简化计算，多刷题总结题型套路，理清三大变换之间的关联。
